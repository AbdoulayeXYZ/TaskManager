# Documentation

## Frontend
The frontend of the application is built using Angular.
To install the Angular CLI, open a terminal window and run the following command:
##  [Angular](https://angular.io/)
    `npm install -g @angular/cli`
    
To install the dependencies, open a terminal window and run the following command:
`npm install`

You can start the development server by running the following command:
`ng serve`

This will start the development server on port 4200. You should be able to access the app in your browser at 
`http://localhost:4200`

## Backend
The backend of the application is built using:
##  [Node js](https://nodejs.org/)
    `npm i node` 
##  [Express](https://www.npmjs.com/package/express)
    `npm i express` 
## [Sequelize](https://sequelize.org/)
    `npm i sequelize` 
## [Typescript](https://www.typescriptlang.org/)
    `npm i typescript` 

To install the dependencies, open a terminal window and run the following command:
`npm install`

Start the server with the following command:

`npm start`

The server listens for requests on port 3000. You should be able to access the app in your browser at 
`http://localhost:3000`
